# End-to-End Marketing Funnel & Conversion Analysis
##  Overview
Analyzed user journey across the marketing funnel (**Visitor → Signup → Purchase**) to identify drop-offs, measure conversion rates, and evaluate channel performance.

## Business Problem
Only a small percentage of users convert into customers.
The goal was to identify **where users drop off and which channels drive high-quality conversions**.

## Tools Used
* SQL (CTEs, Aggregations)
* Power BI (Dashboard, KPI tracking)
* Excel / CSV
  
## Dataset

* 1,000+ user records
* Fields: user_id, source, campaign, stage, date
  
## Key Analysis
* Funnel conversion & drop-off rates
* Channel-wise performance (Google, Meta, YouTube)
* Campaign effectiveness
* Daily conversion trends
* Retention analysis
* User journey (time to purchase)
  
## Key Insights
* Only **~9% users converted overall**
* **66% drop-off at Visitor → Signup stage** (major bottleneck)
* **Google generated highest traffic (~480 users)** but lower conversion
* **YouTube showed highest conversion efficiency (~38%)**
* Meta delivered moderate performance (~24–37%)
* Returning users showed higher likelihood of conversion
  
## Business Impact

* Identified key funnel bottleneck affecting conversions
* Recommended improving signup experience to increase conversion rate
* Suggested shifting budget toward high-converting channels (YouTube)
* Enabled data-driven marketing optimization

## Skills Demonstrated
SQL • Data Analysis • Funnel Analysis • Marketing Analytics • Power BI • Business Insights

## Conclusion
This project highlights how analyzing funnel data can uncover conversion issues and improve marketing performance through data-driven decisions.
